Name

gluErrorString - produce an error string from a OpenGL ES or GLU error code

C Specification

const GLubyte* gluErrorString(GLenum  error);

Parameters

error

Specifies a OpenGL ES or GLU error code.

Description

gluErrorString produces an error string from a OpenGL ES or GLU error code. The string is in ISO Latin 1 format. For example, gluErrorString(GLU_OUT_OF_MEMORY) returns the string out of memory.

The standard GLU error codes are GLU_INVALID_ENUM, GLU_INVALID_VALUE, and GLU_OUT_OF_MEMORY. Certain other GLU functions can return specialized error codes through callbacks. See the glGetError reference page for the list of OpenGL ES error codes.

Errors

NULL is returned if error is not a valid OpenGL ES or GLU error code.
